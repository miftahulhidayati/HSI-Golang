# Mari berlatih

Buat file `main.go` dengan isi berikut:

```go
package main

import "fmt"

func main() {
    // Buatlah sebuah program console, untuk menampilkan biodata antum/na sebagai berikut
    // 
    // Nama : Darizki Pratama
    // NIM  : ARN161-0431
    // Domisili : Jakarta
    // 
    // Dan Tampilkan tulisan berikut
    // Bismillah, semoga Allah ta'ala mudahkan
}
```

Jalankan program menggunakan perintah `go run main.go`. 
Jika sudah silahkan antum/na upload screen-capture di grup channel Latihan dan Tugas di Telegram

Selamat belajar!
